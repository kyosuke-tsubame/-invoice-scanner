# 納品書スキャナー アプリ 引き継ぎドキュメント

## プロジェクト概要
納品書の写真を撮影してOCRでデータ化し、Googleスプレッドシートに自動記録するWebアプリ。

## 今後やること（優先順）

### 1. 課金設定（最優先）
Google Cloud Vision APIを動かすためにクレジットカード登録が必要。
無料枠内（月1,000枚）なら請求なし。

### 2. Google Sheets API連携
スキャンしたデータをGoogleスプレッドシートに自動追記する。

**スプレッドシートの構成：**
| 列 | 内容 |
|----|------|
| A | 日付 |
| B | 店舗名 |
| C | 仕入先 |
| D | 商品名 |
| E | 数量 |
| F | 単位 |
| G | 単価 |
| H | 金額 |
| I | 合計 |

**必要な設定：**
- Google Cloud で Sheets API を有効化
- OAuthまたはAPIキーで認証
- スプレッドシートIDを取得

### 3. アプリUIの改修
現在のindex.htmlを以下の仕様に更新：

**アップロード時の入力項目：**
- 店舗名（選択式・5店舗から選ぶ）
- 納品書写真（複数枚対応）

**読み取り後の動作：**
- OCR結果を画面に表示（日付・取引先・金額を自動抽出）
- 内容を確認・手動修正できる
- 1枚の納品書に複数店舗分が混在する場合は、確認画面で店舗を変更できる
- 「スプレッドシートに保存」ボタンで送信
- 確認後にスプシへ追記

**CSVダウンロードは廃止**（スプシに一本化）

## スプレッドシート構成

月ごとにシートを分けない。全データを1シートに縦積みして、ピボットテーブルで集計する。

| シート名 | 役割 |
|----------|------|
| データ | アプリが追記する生データ（日付・店舗名・取引先・金額） |
| 集計 | ピボットテーブルで月別・店舗別・取引先別を自動集計 |

集計シートは最初に一度設定すれば、以降はデータ追記のたびに自動更新される。
